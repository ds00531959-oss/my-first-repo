# Selenium Automation Framework

## 📌 Project Overview
This project is an automation testing framework built using Selenium WebDriver, Python, and PyTest.  
It automates key functionalities of a web application to ensure reliability and reduce manual testing effort.

---

## 🚀 Test Scenarios Covered
- Login functionality testing
- Dashboard validation
- Advance filter testing

---

## 🛠 Tech Stack
- Python
- Selenium WebDriver
- PyTest
- pytest-html (for reporting)
